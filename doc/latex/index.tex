\hypertarget{index_autotoc_md0}{}\doxysection{Compare\+Commits}\label{index_autotoc_md0}
\mbox{\hyperlink{classCompareCommits}{Compare\+Commits}} produces distance matricies in N\+E\+X\+US file format from commits given by the user.

Look how easy it is to use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ ./comparecommits -\/r './mineserver' mineserver}
\end{DoxyCode}
\hypertarget{index_autotoc_md1}{}\doxysubsection{Features}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Compare all commits of given repository
\item Compare all zips in a given directory
\item Compare commits given in a text file of given repository
\item Compare commits between the start and end commit of a given repository
\item Compare commits of a given vector of commit hashes
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{Installation}\label{index_autotoc_md2}
